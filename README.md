# ☕ SkillSwap Coffee

**Trade your talent for knowledge.**  
A premium, minimalist marketplace designed for campus experts to exchange skills over a cup of coffee.

---

## 🌟 Overview
SkillSwap Coffee is a "Software-over-Hardware" initiative aimed at fostering a collaborative learning culture within university campuses. Instead of a traditional currency, students trade their expertise—be it coding, design, or physics—for a mentorship session.

This project focuses on **Social Architecture**, **Data Persistence**, and **Premium UI/UX**.

## 🎨 Design Philosophy
The interface follows a **High-End "Coffee Shop" Aesthetic**:
*   **Palette:** Beige (`#F5F5DC`), Deep Espresso (`#3E2723`), and Soft Cream (`#FFFDD0`).
*   **Style:** Minimalist Brutalism with thick borders and 8px offset shadows.
*   **Vibe:** Professional, warm, and structured.

## 🚀 Features
*   **Skill Menu (Browse):** View a dynamic grid of available mentors and their "teaching vibes."
*   **Mentor Registry (Offer):** A simple, validation-ready form to list your own skills.
*   **Local Persistence:** Uses `localStorage` to save user data directly in the browser—no backend setup required for the prototype.
*   **Responsive Layout:** Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Technical Stack
*   **Structure:** Semantic HTML5
*   **Styling:** CSS3 (Custom Properties & Grid Layout)
*   **Logic:** Vanilla JavaScript (ES6+)
*   **Icons:** Lucide-inspired SVG patterns

## 📦 Getting Started
1.  **Clone/Download:** Download the `index.html` file.
2.  **Open:** Simply double-click `index.html` to run it in any modern web browser.
3.  **Deploy:** Upload to **GitHub Pages** or **Netlify** for a live URL in seconds.
